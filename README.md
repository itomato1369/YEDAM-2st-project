# 2. 프로젝트 개요
* MES에 대한 공정과 용어 이해. 생산관리 시스템을 데이터화 하는 것에 대한 구현 중심 프로젝트
  
  요구사항 분석, 화면설계, ERD설계 등 미리 작성된 서류를 참고하여 실제 화면과
  
  기능을 구현하는 것에 집중하였습니다.
